Uloženie zmien

Ako získať token?

1. GitHub → Settings → Developer settings → Fine-grained personal access tokens → Generate new token.
2. Repository access: salqen/cafedelparadiseweb. Permissions: Contents → Read and write.
3. Token vlož sem. Po uložení sa zmena commitne do repozitára a GitHub Pages web sa automaticky obnoví (1–2 min).
Bez tokenu môžeš použiť „Stiahnuť cola.json" a súbor nahrať do data/cola.json ručne.
