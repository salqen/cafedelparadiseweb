Ako získať token?

1. GitHub → Settings → Developer settings → Fine-grained personal access tokens → Generate new token.
2. Repository access: salqen/cafedelparadiseweb. Permissions: Contents → Read and write.
3. Token vlož sem. Po uložení sa zmeny commitnú a GitHub Pages web sa obnoví (1–2 min).
Bez tokenu použi „Stiahnuť JSON súbory" a nahraj ich do data/ ručne.
